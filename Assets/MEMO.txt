    - 데이터

        - 보유한 아이템, 업그레이드 현황, 사용할 타워 종류와 같은 유저 데이터

잡소리
    - 메뉴 유아이 매니저
        - 게임 전체의 ui 를 관리하는 매니저
        - 하지만 can be destoryed on load -> 싱글톤이 아님!
        - 모든 scene 에 배치
            - scene 의 정보를 읽어옴
            - 읽어온 scene 에 필요한 ui 가 있는지만 체크하는 식으로 동작하기
    - "게임매니저" 외 군소 매니저들
        - 싱글톤이 아님
            - 중요한 정보는 게임 or 데이터 매니저를 통해 r/w 하기
    - 게임매니저
        - FindManagerOnSceneLoad(string sceneName) 같은 메서드를 추가할 것
